#  Personal Budget Project

## Version JDK and Data base

- Java: 16
- Mysql

### How execute the project in Linux
### Prerequisites

1. Clone the project:
   ```https://github.com/andrericardoalves/personalbudget-api```

### Running the unit tests
1.Enter the project directory and run:
```./mvnw test```

### Running the project
1. Enter the project directory and run:
```mvn spring-boot:run```
